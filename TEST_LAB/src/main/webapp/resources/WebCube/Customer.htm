Teruten’s WebCube is a web browser security solution that prevents the illegal use of web site’s diverse constituent elements.
 
Control of web browser’s editing functions
- Control of the function of save the web page as other names
- Control of print and clipboard functions
- Control of context menu such as view source, etc.
Control of web browser editing function
- Control of screen capture by using print screen
- Control of the operation of screen capture tools
Protection of the temporarily saved file
- Control of the access to the files temporarily saved on the user’s PC such as image files, flash files,
source files, etc.
 
 
Teruten’s WebCube easily protects web sites by installing WebCube.ocx.
 
 
 
Teruten’s WebCube protects all constituent elements included in web sites regardless of their
types.
- The protection applies to all elements that constitute web pages such as texts, image files, etc.
- Unauthorized outflow of flash files (swf) is prevented.
- The protection applies to compound contents such as flash + image + test contents without restrictions.
- Selective security can be applied to the constituent elements.
Whether to apply security to text, image files, etc. can be set by parameters.
- Access can be controlled per user authority.
Constituent element usage authority is given according to the service user’s authority level.
Teruten’s WebCube can prevent the illegal use of web sites at a minimum expense.
- The hardware expense for DRM server, etc. is not necessary.
- No human/time expense is necessary for content packaging, etc.
- With the realization of a simple application method, no additional management labor is necessary.
Only the object creation scripts are added to the established web page sources.
